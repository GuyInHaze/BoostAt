# Проект: BoostAT

## Технологии, которые использовались

### Figma:
- [x] Построение Сетки
- [x] Создание Стилей
- [x] Создание Компонентов
- [x] Анимации в Figma
- [x] Редактирование Шрифтов
- [x] Создание UI kit
- [x] Адаптацию макета под разные типы устройств- (1280\1024\768\480\320

### HTML:

- [x] Pixel Perfect Верстка

### CSS:
